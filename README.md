This project demonstrates how to run Playwright tests that parse content from a PDF file and perform assertions on the extracted text.

# Prerequisites <br>
Node.js (v18 or above recommended) <br>
npm

# Installation
1️⃣ Initialize the project <br>
npm init -y <br>

2️⃣ Install Playwright <br>
npm install -D @playwright/test <br>

3️⃣ Install PDF parsing library <br>
npm install pdf-parse <br>

▶️ Run the Tests <br>
npx playwright test
